
argument可能存在其他参数，此文档仅说明样式配置参数types
例：<list_1 types="1"></list_1>

场景配置：
	模块名：list_1
	arguments:{
		types：分五种 （1,2,3,6,9）
	}
套系配置：
	模块名：recommend
	arguments:{
		types:分三种 （1,2,3）
	}
导航模块：
	模块名：nav
	arguments:{
		types:分三种 （1,2,3）
	}
标题模块：
	模块名：title
	argument:{
		types:分三种 （1,2,3）
	}
线下资讯模块：
	模块名：service_offline
	argument:{
		types:分三种 （1,2,3）
	}
